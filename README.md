# Retail Sales Performance Dashboard (NSW & VIC, 2018–2022)

       Project Overview
This project analyzes retail sales data across New South Wales (NSW) and Victoria (VIC) over a 5-year period (2018–2022). The goal is to evaluate revenue, cost, profit, customer segments, and regional performance.

        Tools Used
- Excel (Data Cleaning)
- SQL Server (Data Storage & Analysis)
- Power BI (Dashboard & Visualization)
       Key Features
- Calculated KPIs: Revenue, Cost, Profit, Total Orders
- Sales trend analysis over time (2018–2022)
- Revenue breakdown by product category and state
- Customer segmentation analysis
- Shipping performance evaluation

        Key Insights
- NSW generates higher revenue than VIC
- Sales show steady growth from 2018 to 2022
- Corporate customers contribute the most revenue
- Express shipping delivers faster than standard shipping

         Project Workflow
1. Data cleaning in Excel / sql  
2. Data loaded into SQL Server  
3. SQL queries used for KPI calculations  
4. Dashboard built in Power BI
